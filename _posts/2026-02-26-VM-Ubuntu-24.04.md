# VirtualBox Ubuntu 24.04 LTS Dev VM

## VM Configuration
1. RAM: 16 GB
2. CPU: 4 cores
3. Disk: 200 GB
4. Graphics Controller: VMSVGA
5. Video Memory: 128 MB
6. 3D Acceleration: Initially enabled, later disabled for stability with Xorg
7. Network Adapter: Bridged Adapter (for LAN access)

These settings provide optimal performance for Ubuntu dev environment, balancing speed for the VM while keeping the host system responsive.

## Ubuntu Installation

ISO Used: Ubuntu 24.04 LTS,Desktop Environment: GNOME.
Why GNOME over KDE:
- More stable in VirtualBox
- Less issues with 3D acceleration and Wayland
- Works smoothly with full-screen and auto-resize

If you want KDE Plasma look, you can install Kubuntu, but it may cause crashes or display issues in VirtualBox.

## Guest Additions Installation
1. Insert Guest Additions CD:
- VirtualBox menu: Devices → Insert Guest Additions CD Image
- Download if prompted

2. Install Dependencies:

```bash
    sudo apt update
    sudo apt install build-essential dkms linux-headers-$(uname -r)
```

3. Run Guest Additions Installer:

```bash
    cd /media/$USER/VBox_GAs_*
    sudo sh VBoxLinuxAdditions.run
```

4. Rebuild Guest Additions (if needed):

```bash
    sudo /sbin/rcvboxadd setup
```

5. Restart VM:

```bash
    sudo reboot
```

6. Verification:

```bash 
    lsmod | grep vbox
```

Expected output after successful installation:
- vboxguest → core module
- vboxvideo → video module (required for auto-resize/full-screen)
- vboxsf → shared folders (optional)


## Common Issues and Solutions

| Issue                                  | Cause                               | Solution                                                    |
|----------------------------------------|-------------------------------------|-------------------------------------------------------------|
| Screen stuck at 1280x800               | `vboxvideo` module not loaded       | Reinstall Guest Additions and rebuild kernel modules        |
| Login session crash / returns to login | Wayland + 3D Acceleration           | Use Xorg session and disable 3D acceleration in VM settings |
| Full-screen not working                | Auto-resize disabled or GA missing  | Enable Auto-resize and reinstall Guest Additions            |
